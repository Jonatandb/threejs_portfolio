## Run Locally

Clone the project

```bash
  git clone https://github.com/Jonatandb/threejs_portfolio
```

Go to the project directory

```bash
  cd threejs_portfolio
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```

Visit:
- http://localhost:5173
